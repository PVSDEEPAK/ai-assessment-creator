# AI Assessment Creator

AI Assessment Creator is a full-stack AI-powered educational platform designed to help teachers and educators generate assignments automatically from uploaded study material.

The platform analyzes uploaded PDF notes, extracts educational content, and generates structured assessments using AI models. It also provides assignment management, dashboard functionality, and cloud deployment support.

---

# Features

* AI-powered assignment generation
* Upload PDF study material
* Automatic question paper generation
* Responsive modern dashboard UI
* Assignment history management
* MongoDB database integration
* REST API backend architecture
* Cloud deployment using Vercel and Render
* Full-stack TypeScript implementation

# Project Workflow

1. Upload educational PDF material.
2. Backend extracts text from uploaded files.
3. AI model analyzes the content.
4. Structured assignments are generated automatically.
5. Assignments are stored in MongoDB.
6. Teachers can manage assignments through the dashboard.

# Installation & Setup

## Clone Repository

```bash
git clone https://github.com/PVSDEEPAK/ai-assessment-creator.git
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## Backend Setup

```bash
cd backend
npm install
npm run dev
```

---

# Environment Variables

Create a `.env` file inside backend:

```env
MONGODB_URI=your_mongodb_uri
OPENROUTER_API_KEY=your_api_key
PORT=5000
```
